POSITION: STATIC

Il valore "static" è il valore di default della proprietà position. Un elemento dotato di "position: static" non è posizionato nel documento in nessun modo speciale, seguirà il normale flow degli elementi. Si dichiara esplicitamente quando c'è bisogno di "resettare" il comportamento di un elemento, magari dopo che un altra regola css ne ha alterato la funzionalità.
